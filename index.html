Mission Control

Your open tabs, organized by workspace.

Every open page is grouped by domain into a scannable card. You see what you actually have open — not just a long flat list.

  • Groups tabs by domain; filters out chrome://, about:, and browser internals
  • Shows duplicate URL count per group; one click closes all extras
  • Close a single tab, close a group, or close everything except pinned and active tabs
  • Create a native Chrome tab group per domain — named, colored, and collapsed automatically
  • Tab Spaces classify open pages into Study, Work, Play, and Other — open a space in a new focused window
  • Launchpad shortcuts: ChatGPT, Notion, GitHub, Gmail, Drive, Calendar, Reader guide

Workspace Tasks

Remember why you opened those tabs.

Each domain workspace can hold lightweight tasks. When you try to close a workspace that still has open items, Tabspace pauses and asks what you want to do instead of closing silently.

  • Add tasks directly on any workspace card — they stay attached to that domain across sessions
  • Click a task to toggle it done; shows up to 2 tasks inline with a count for the rest
  • Closing a workspace with open tasks opens a modal: finish one and close, defer and close, or keep it open
  • Tasks persist in Chrome local storage — survive new tab refreshes

Reading Queue & Local Note

Save it now. Come back when you're ready.

The side rail keeps two small utilities beside your tab workspace — a read-later queue and a scratchpad that saves directly to your disk.

  • Reading Queue — save any tab for later from its workspace card; duplicate unfinished URLs are blocked automatically
  • Mark queue items done or remove them; completed items are hidden from the active list
  • Local Note — type or paste text, set a filename, save as a .txt file directly to your computer
  • Uses the browser's file picker when available; falls back to Chrome downloads API otherwise

Reference Board

Bookmarks as a searchable reference shelf.

Your Chrome bookmark folders become scannable reference cards with a live search across title, URL, domain, and folder path. Organize mode lets you clean up without leaving the new tab.

  • Reads your full bookmark tree; displays each folder as a collapsible card
  • Search across title, URL, hostname, and folder path simultaneously
  • Organize mode — select bookmarks with checkboxes, move to an existing or new folder, or delete after confirmation
  • Duplicate Detector — finds same-URL duplicates and same-domain/same-title near-duplicates; select extras in one click
  • Unfiled Cleanup — suggests a domain-based folder for loose bookmarks; one click moves them all

Reader Assist

Make any article easier to read and annotate.

A content script injected into every webpage adds a lightweight reading layer — no page modifications, no data sent anywhere. Activate it from the Tabspace launchpad guide.

  • Focus Line — highlights your current reading block; surrounding content fades back
  • Navigate blocks with J / K or arrow keys; press Esc to exit
  • Annotations — select text to get a toolbar: Yellow, Concept, Action, Question, Skim, Underline
  • Annotation shortcuts: Y Yellow · B Concept · A Action · Q Question · S Skim · U Underline
  • Highlights saved to Chrome local storage and restored on the same URL next visit
  • 15-minute reading timer included

Local-first

Everything stays in your browser.

No remote server

Tabspace never sends your tabs, bookmarks, notes, annotations, or browsing activity anywhere.

Chrome local storage only

Reading queue, workspace tasks, and Reader Assist annotations are stored in Chrome extension-local storage on your machine.

No ads, no analytics

No ad network, no usage telemetry, no resale of browsing data. The extension has no network requests of its own.

Permissions explained

tabs + tabGroups to read and organize open tabs. bookmarks to display and edit your saved bookmarks. storage for queue and annotations. downloads for Local Note file saving.
